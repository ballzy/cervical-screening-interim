Pagination - when not needed

If we know there are going to be less than 50 results on page then pagination is not required.

Pagination - when required

If we are unsure on the amount of rows of data then pagination can be added.

The simplest way is to use the pagination component from the service manual.

Multple downloads options

For multiple buttons, determine if there is a primary action and make this green, other buttons can be grey
